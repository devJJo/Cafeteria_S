# Cafeteria_S 
Cafeteria App 서버부분 사용 언어 및 기술  

  
특정 지역(현재 구로디지털단지)근처의 구내식당 정보를  
제공해주는 API를 제공하고 관리자 페이지로 식당 리스트 및 메뉴를  
입력할 수 있는 화면을 위한 프로젝트


**-서버**  
AWS - ec2 (unbuntu 18.04)  
Tomcat8  
Apache    
  
**-Language**  
Java  
Jsp/Javascript/jQuery    
  
**-DB**  
PostgreSQL 10.6    
  
**-Framework**  
Spring Framework 4.3    
  
**-IDE**  
STS    
  
**-ETC**  
Mybatis  
Restful Api
